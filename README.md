## Sobre
Gerenciador de compras CLI simples e eficiente, desenvolvido em Golang

## Instalação
Clone este repositório:
``` sh
git clone https://github.com/Random767/shopping-list
```

Entre no diretório do projeto:
``` sh
cd shopping-list
```

Compile o projeto:
``` sh
go build
```

Execute o programa:
``` sh
./shopping-list
```
